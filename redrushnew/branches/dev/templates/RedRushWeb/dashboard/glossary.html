GLOSSARY

Visit:

Total number of visits to the website based on the data imported from Google Analytics

Time On Site:

Visit length of all visitors based on the data imported from Google Analytics

User:

Total number of unique users logged in to the website

Participant:

Total number of level 1 users that engaged in any one of the RedRush online activities (race, buy parts, customize car or mini games)

Login:

Total number of unique users logged in to the website

ACTIVE PARTICIPANT :

Total number of level 1 users that engaged in any one of the RedRush online activities (race, buy parts, customize car or mini games)

ACTIVE PARTICIPANT/LOGIN:

The sum of the number of Participants divided by the number of Users

Loyalty:

returning user to the site

Average Page Views:

The total time spent on a page divided by the number of unique visits to that page. The value is shown in minutes

Page View Distribution:

The distribution of the number of page views viewed by users on each page of the website

ACTIVITY Distribution:

The distribution of the number of activities engaged by users on each activity page of the website

Geographical Distribution:

The distribution of the number of users from individual cities based on the MOP registration data.

Average Time on Activity Ditribution:

The distribution of the average time spent on each RedRush activities

Average RacesPlayed:

The average number of races played by users each day

Average Car Modifications:

The average number of car modifications have been made by users each day

Average Time on Each Level:

The average time spent by users on each level of racing

Total mini game played:

Total number of mini games played by users

Percentage Plays by Game:

Comparative percentages of the number of mini games played by users

Average time on game: :

Comparative chart of the average time spent by users on each game

Merchendise Redeemed:

Comparative fluctuations of the total number of each merchandise redeemed by users

Device Used:

Comparative percentages of the devices used to access the website

Gender/Age:

Total number of users based on age and gender classification

Brand Preference:

Comparative percentages of users preferred brands
